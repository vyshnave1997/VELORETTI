# VELORETTI

A modern React web application for bike lease plans, featuring a clean UI, responsive design, and interactive components.

## Features

- Bike lease plans for employees
- Product showcase with color and image selection
- Service, maintenance, and insurance information
- Step-by-step guide on how it works
- Contact form and FAQ section
- Stylish, responsive design using SCSS

## Getting Started

1. Clone the repository:
   ```sh
   git clone https://github.com/your-username/veloretti.git
   ```
2. Install dependencies:
   ```sh
   npm install
   ```
3. Start the development server:
   ```sh
   npm start
   ```
